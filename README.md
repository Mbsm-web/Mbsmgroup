== Description ==
Professional static HTML generation tool by MBSM Group. Automatically converts your WordPress posts into portable .html and .htm files for lightning-fast loading, improved SEO, and offline reading.

== Features ==
*   Automatic export on "Save Post".
*   Bulk export for existing content.
*   Built-in PDF embedder for static pages.
*   Sidebar metabox for quick link access.
*   Responsive table fix for static versions.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate via the 'Plugins' menu.
3. Ensure the folder `/html/` in your WordPress root is writable (755 permissions).
4. Go to **Mbsmpro Html** menu to configure your File Prefix and AdSense codes.

== Usage ==
Navigate to the "Bulk Export" tab to generate files for all your existing published posts at once.
